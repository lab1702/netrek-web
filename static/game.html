NETREK CONTROLS

MOVEMENT

0-9 Set speed (0=stop, 9=max)
!, @, # Set speed 10, 11, 12 (Shift+1,2,3)
Right Click Set course/direction

COMBAT

Left Click Fire torpedo
Middle Click Fire phaser
p Fire plasma torpedo
d Detonate own torpedoes

SHIP SYSTEMS

s Toggle shields
c Toggle cloak
r/R Toggle repair mode
t Tractor beam nearest enemy
y Pressor beam nearest enemy

PLANETS

o Orbit planet
b Bomb planet (when orbiting)
z Beam up armies
x Beam down armies

TARGETING

l Lock on nearest object to cursor
i/I Show info for object under cursor

COMMUNICATION

a Send message to all
/ Quick slash commands (opens All message with /)
t/T Send message to team (Shift+t or T)
Escape Cancel message

OTHER

\ Toggle practice panel
? Toggle this help window
Q Quit/Self-destruct (capital Q)
