NETREK-WEB

Web Implementation of the Classic Internet Space Battle Game

⚠️ PROOF OF CONCEPT - Work in Progress ⚠️

Launch Game

Essential Controls

Mouse Commands

  • Left Click Fire Torpedo
  • Middle Click Fire Phaser
  • Right Click Set Course

Keyboard Commands

  • 0-9 Set Speed
  • S Shields
  • C Cloak
  • ? Help

This interpretation of Netrek was created by lab1702, who used to play from the MTEK lab at the Chalmers University of Technology, Gothenburg, Sweden in the early 90s.

Tech: server written in Go, client written in vanilla JavaScript, communication via WebSockets.

All code generated with the help of AI coding assistants as a way for me to learn how to use AI.
