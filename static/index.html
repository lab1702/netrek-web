Current Features (Proof of Concept)

  • Core NETREK gameplay mechanics
  • Real-time multiplayer space combat
  • Seven ship types: Scout, Destroyer, Cruiser, Battleship, Assault, Starbase, Galaxy
  • Team-based warfare with planet conquest
  • Practice mode with AI bots for testing
  • Physics simulation with turn rates and momentum
  • Cloaking, shields, tractors, and repair systems
  • Lock-on targeting for navigation and combat
  • Tactical and galactic map views

NETREK-WEB - A modern web implementation of the classic game

This is a proof-of-concept version. Expect bugs and incomplete features.

NETREK-WEB built with Go and WebSockets
